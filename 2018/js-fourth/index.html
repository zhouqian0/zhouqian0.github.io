中文排序

  1. 如果直接是中文数组，可以采用 arr.sort() 排序。

  2. 如果对对象数组中的某个属性进行中文排序，可用 js 的 localeCompare() 对中文排序。

    corps.sort((a, b) => {
  return a.shortName.localeCompare(b.shortName, 'zh-Hans-CN',{sensitivity:'accent'});
});
    • 不同浏览器可能会出现不同结果
    • 无法识别多音字
