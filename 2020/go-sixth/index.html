Go 学习小记1

本文将介绍 golang 的优势，基本类型和常量…

Go is a strongly typed language.

Advantages of golang

  1. Simple syntax
  2. Easy to write concurrent programs
  3. Compiled language
  4. Fast compilation
  5. Static linking
  6. Go Tooling
  7. Garbage collection
  8. Simple language specification
  9. Opensource

Basic types available in golang

  1. bool
  2. Numeric Types
    • int
      • int8 (-128 to 127)
      • int16 (-32768 to 32767)
      • int32 (-2147483648 to 2147483647)
      • int64 (-9223372036854775808 to 9223372036854775807),
      • int (the size of int is 32 bit in 32 bit systems and 64 bit in 64 bit systems)
    • uint
      • uint8 (0 to 255)
      • uint16 (0 to 65535)
      • uint32 (0 to 4294967295)
      • uint64 (0 to 18446744073709551615)
      • uint (the size of uint is 32 bit in 32 bit systems and 64 bit in 64 bit systems)
    • float
      • float32
      • float64
    • complex
      • complex64 (float32 real and imaginary parts, both the real and imaginary parts must be of the same type)
        • complex(5, 7) = 5 + 7i
      • complex128 (float64 real and imaginary parts, both the real and imaginary parts must be of the same type)
    • byte (alias of uint8)
    • rune (alias of int32)
  3. string (collection of bytes)

Type convert in golang

  1. Go is very strict about explicit typing.
  2. T(v) is the syntax to convert a value v to type T.

Constants in golang

  1. The value of a constant should be known at compile time. Hence it cannot be assigned to a value returned by a function call since the function call takes place at run time.
  2. Any value enclosed between double quotes is a string constant in Go.
  3. The constant still doesn’t have a type.
    • The untyped constants have a default type associated with them and they supply it if and only if a line of code demands it.
