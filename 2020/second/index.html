Vim 操作

命令行键入 vimtutor zh 阅读中文教程

  1. 移动 
            ^
        k            
  < h       l >             
        j                  
        v
  2. 删除：
    • x 删除光标位置字符
    • dw 从当前光标当前位置直到下一个单词起始处，不包括它的第一个字符
      • “hello world” 光标选中 h，则删除后光标选中 w
    • de 从当前光标当前位置直到单词末尾，包括最后一个字符
      • “hello world” 光标选中 h，则删除后光标选中 w 前面的空格
    • d$ 删除当前光标所在行以光标位置开头的的行后的所有内容
    • dd 删除整行
  3. 插入：
    • a 光标后插入字符
    • i 光标前插入字符
    • A 在光标行所在的末尾插入字符
  4. 使光标回到行首
    • 0 光标选中行内任意字符，键入 0 则光标移动到行首
  5. motion 操作对象
    • w 光标转跳到下一个单词的开头
    • e 光标转跳到当前单词的结尾
    • $ 光标转跳到当前行的结尾
    • 数字加上 motion 表示执行的次数，如 2w 表示执行两次 w
    • 行为 + 数字 + 操作对象
      • d2w，表示执行两次 dw
      • 2dd，表示删除当前行开始的两行
  6. 撤销
    • u 撤销上次操作
    • U 撤销在当前行的所有操作
    • CTRL-R，撤消以前的撤消命令
  7. 置入
    • p 重新置入已经删除（d）/复制（y）的文本内容
  8. 替换
    • r 在光标所在处键入 r，再键入目标字符，可将光标处的字符替换为目标字符（单个字符）
    • R 在光标所在处键入 R，可从光标处练习替换多个字母，退格则会撤销上一个修改，按 esc 退出
  9. 更改
    • c
      • cw 更改光标处开始到下个单词开始处的文本，键入后需要再输入目标字符
      • ce 更改光标处开始到当前单词结尾处的文本，键入后需要再输入目标字符
      • 可组合 motion 和 number
        • c$
        • c2w
        • 键入后会删除被选中的内容
  10. 定位及文件状态
    • CTRL-G 显示的内容是当前编辑的文件名和文件中光标位置
    • G 当前光标直接跳转到文件最后一行
    • gg 使当前光标直接跳转到文件第一行
    • number G 使光标调到行号为 number 的行
  11. 搜索
    • /target 向下查找 target 字符
      • /\ctarget 不忽略大小写查找 target（单次查找）
      • /\Ctarget 忽略大小写查找 target（单次查找）
    • ？target 向上查找 target 字符
    • n 跳转到上一次 target 字符的位置
    • N 跳转到相反方向查找同上一次的字 target 字符的位置
    • CTRL-O 回退到上一次的查找
    • CTRL-I 跳转到较新的位置
  12. 查找配对括号
    • % 光标选中 )、]、}，查找配对的括号
  13. 批量替换
    • :s/old/new 将替换 old 为 new，只改变光标所在行的第一个匹配串
    • :s/old/new/g 替换光标所在行全行的匹配串
    • #,#s/old/new/g 替换制定行匹配串，其中 #,# 代表的是替换操作的若干行中首尾两行的行号。
    • %s/old/new/g 替换整个文件中的每个匹配串。
    • %s/old/new/gc 找到整个文件中的每个匹配串，并且对每个匹配串提示是否进行替换。
  14. 执行外部命令
    • :! 键入:!，然后紧接着输入一个外部命令可以执行该外部命令
  15. 保存文件
    • w filename 将更改保存到另一个文件
    • 键入 v， 移动光标可高亮选中行，然后键入 : ，会发现底部出现:'<,'>，然后键入 w filename，可讲选中内容保存到 filename 中
    • :r FILENAME 向当前文件中插入另外的文件的内容
    • :r !ls 可以读取 ls 命令的输出，并把它放置在光标下面（同理可读取外部命令的输出）
  16. 打开类命令
    • o 在当前光标行下插入一个新行，并进入编辑模式
    • O 在当前光标行上插入一个新行，并进入编辑模式
  17. 复制文本
    • y 用v高亮目标文本，键入 y，则可复制内容，再键入 p 实现复制粘贴
    • 可使用 yw 复制单词
  18. 设置类命令
    • :set ic 忽略大小写(Ignore Case，忽略大小写)
    • :set noic 禁用忽略大小写
    • :set hls is 高亮搜索结果
    • :nohlsearch 解除高亮搜索结果 = :set nohls
    • no + 设置选项可关闭之前开启的选项
