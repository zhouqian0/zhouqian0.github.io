.every() Method

arr.every(fn)

arr.every(fn) 这个函数可以快速验证数组是否满足指定的校验函数，而不需要去遍历数组结构

const nums = [5];
console.log(nums.every((n)=> n === 5)); // true
console.log(nums.every((n)=> n === 4)); // false

nums.push(10)
console.log(nums.every((n)=> n % 5 === 0)); // true

但是这个函数也存在一定的缺陷，那就是: 当 target arr 是一个空数组时，不管是什么校验函数，它都将返回 true 。 所以使用这个函数需要先对 target arr 的长度做一个简单的判断。

const nums = [];
console.log(nums.every((n)=> n === 5)); // true
console.log(nums.every((n)=> n === 4)); // true

// -----------------
let res;
if(nums.length > 0){
  res = nums.every((n)=> n === 5);
}else{
  res = false;
}
