angular datepicker 使用建议

datepicker

实现点击 input 后会打开日期选择器：由于input标签可点击，可能会触发浏览器的自动联想（autocomplete）,会出现： 先打开可自动联想的下拉框，然后该下拉框消失，出现日期选择 。针对于此，现解决方案为禁用日期选择器的 input 标签的 autocomplete 属性（autocomplete=‘off’）,然后添加 click 事件 (click)=“picker.open()“。

<mat-form-field>
  <label>
    <input matInput [matDatepicker]="picker" autocomplete="off" required formControlName="date"
           placeholder="选择日期" (click)="picker.open()" [errorStateMatcher]="matcher">
  </label>
  <mat-datepicker-toggle matSuffix [for]="picker"></mat-datepicker-toggle>
  <mat-datepicker #picker></mat-datepicker>
</mat-form-field>
