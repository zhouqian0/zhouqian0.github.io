异步方法中的同步

await

  1. 通过将 observable 转换为 promise，调用 promise 的 then() 方法，达到同步效果（在异步函数中）。

  2. 在异步函数中使用同步，需要加上 await 等待 promise 结束后再执行下一步操作。

    getALLReportData() {
  return (this.reportService.getReportBlockData() as Observable).toPromise();
}
    
（await） this.getALLReportData().then(res => this.bkDataList = res['data'] || []);
